# 📅 Schedule Bot — Бот управления графиком работы

## Структура файлов

```
schedule_bot/
├── bot.py          # Точка входа
├── config.py       # Настройки (токен, станции, дни)
├── storage.py      # Хранение сотрудников (JSON файл)
├── scheduler.py    # Генерация расписания
├── renderer.py     # Рендер таблицы в PNG (Pillow)
├── handlers.py     # Все хэндлеры бота
├── requirements.txt
└── data.json       # Создаётся автоматически
```

## Установка и запуск

```bash
cd schedule_bot
pip install -r requirements.txt

# Задать токен бота
export BOT_TOKEN=ваш_токен_от_BotFather

python bot.py
```

## Использование

| Команда | Описание |
|---|---|
| `/start` | Приветствие и список команд |
| `/add_employee` | Добавить сотрудника и выбрать его станции |
| `/employees` | Показать всех сотрудников |
| `/remove_employee` | Удалить сотрудника |
| `/create_schedule` | Создать график на следующую неделю |

## Как работает `/create_schedule`

1. Бот последовательно спрашивает про каждого сотрудника — нажимаете дни-выходные (красные кнопки 🔴)
2. После всех сотрудников — генерируется расписание
3. Бот присылает PNG-картинку таблицы как на скриншоте

## Настройка станций

В `config.py` можно поменять список станций:
```python
STATIONS = ["Кальян", "Бар", "Кухня"]
```

## Алгоритм генерации

- Каждому дню назначается сотрудник на каждую станцию
- Учитываются выходные дни
- Равномерное распределение нагрузки (минимизация смен у одного человека)
- Один сотрудник в один день — одна станция (если хватает людей)
